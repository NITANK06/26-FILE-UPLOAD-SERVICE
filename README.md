# 26-FILE-UPLOAD-SERVICE


A REST API built using Node.js, Express.js, and Multer for uploading, validating, storing, retrieving, downloading, and deleting files.



** Features**

 File Upload API
 File Type Validation
 File Size Restriction
 File Storage Management
 Retrieve Uploaded Files
 Download Files
 Delete Files
 Static File Serving
 Beginner-Friendly Project Structure



 **Technologies Used**

 Node.js
 Express.js
 Multer
 Nodemon
 REST API
 Postman
